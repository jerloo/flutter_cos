# fluent_tencent_cos

一个腾讯云上传插件。

## Getting Started

最近在使用flutter开发APP，用到了腾讯云上传图片的功能。

找到了一个插件 tencent_cos <https://github.com/OpenFlutter/tencent_cos>，加入项目中，发现有bug。
又找了一个插件 fw_cos <https://pub.dev/packages/fw_cos>, 发现依然有bug，并且这俩插件都不支持空安全。
所以我就自己改了一下。

如有涉及侵权，请联系本人进行删除。
